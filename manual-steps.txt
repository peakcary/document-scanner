# 手动部署步骤

## 步骤1: 本地打包文件
cd /Users/peakom
tar -czf document-scanner.tar.gz document-scanner/

## 步骤2: 上传文件（需要输入密码）
scp document-scanner.tar.gz root@47.92.236.28:/tmp/

## 步骤3: 连接服务器（需要输入密码）
ssh root@47.92.236.28

## 步骤4: 在服务器上执行（连接后复制粘贴）
cd /tmp
tar -xzf document-scanner.tar.gz
cd document-scanner
chmod +x deploy.sh
./deploy.sh

## 部署时需要输入的信息：
# 1. 域名：例如 scanner.yourdomain.com
# 2. 邮箱：例如 your-email@example.com

## 注意事项：
# - 需要先购买域名并解析到服务器IP
# - 确保阿里云安全组开放了80和443端口
# - 如果没有域名，可以直接用IP访问（但无法使用HTTPS）
